## 📌 Tentang AssetTrack

**AssetTrack** adalah sistem monitoring aset berbasis web yang dirancang untuk:
- 📍 Melacak lokasi aset secara real-time (GPS)
- 📡 Monitoring koneksi LoRa
- 🔋 Menampilkan status perangkat (baterai, update terakhir)
- 📊 Dashboard interaktif dan responsif (desktop & mobile)

Sistem ini cocok digunakan untuk:
- Kampus (Smart Campus)
- Industri & logistik
- Tracking perangkat IoT
